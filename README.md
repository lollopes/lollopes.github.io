# lollopes.github.io

Google Scholar data is no longer fetched from OpenAlex in the browser.

The site now reads `/data/scholar.json`, which is generated by `scripts/fetch_scholar.py`.

To refresh locally:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements-scholar.txt
python scripts/fetch_scholar.py
```

Optional environment variables:

- `SCHOLAR_USER_ID`: preferred if you know the exact Google Scholar profile ID.
- `SCHOLAR_AUTHOR_NAME`: defaults to `Lorenzo Pes`.
- `SCHOLAR_AFFILIATION_HINT`: defaults to `Eindhoven University of Technology`.

GitHub Actions also refreshes `data/scholar.json` on push, on schedule, or via manual dispatch.
